# lucidUI 4
lucidUI 4 brings several new changes system-wide
## A new name
Hyprlucid is now lucidUI
## Design Upgrades
Parts of the design have been given a facelift
## Redesigned Launcher
The Launcher has been redesigned, bringing a cleaner look
## Improved Animations
The animations are now more fluid and bouncy
## Redesigned lock screen
The lock screen now features a more minimalist look
## Power Options
Pressing CTRL + ALT + DEL at any time reveals the Power Options, showing options like:\
Lock
Logout
Reboot
Power Off
## Install Script
lucidUI can now be installed faster with the new install script
> lucidUI 4.0\
> 1st July 2026

## Hyprlucid 2
Hyprlucid 2 brings several new features to Hyprlucid
### Introducing SwitchWP
SwitchWP is my custom tool for switching wallpapers with ease
It also automatically update Pywal, which will change the look for Wofi, Waybar and SwayNC
### Wofi is now the default launcher
Rofi has been replaced in favour of Wofi\
Wofi allows for Pywal support, along with better support for Wayland\
The Rofi config remains in this repo
### Fullscreen
Press ALT + F to put your current app into fullscreen mode, great for gaming, watching videos or to remove distractions\
