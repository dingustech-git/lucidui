# Add new wallpapers in Hyprlucid 3
This guide will walk you through adding wallpapers in Hyprlucid
## Making sure everything is ready
Find the image you want to set as your wallpaper, make sure it's in the `.png` format\
Rename the image to a number (eg. 6), this is the number you'll put into SwitchWP\
Copy it to ~/Wallpaper/Library/Normal
## Making the blurred version
The blurred version is used by the Lock Screen\
Use my Figma template to add blur\
https://www.figma.com/design/1sGrCoSDbEWHNSyBAKDTS3/Untitled?node-id=0-1&t=LJhwubMkBWLfF2Zx-1 \
Download the image as a PNG\
Copy it to ~/Wallpaper/Library/Blurred
## Setting it as your wallpaper
1. Start SwitchWP
2. Enter the number from earlier
3. After a few seconds, your wallpaper should change
